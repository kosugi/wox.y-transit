# wox.y-transit

is a simple Yahoo! Japan transit launcher with two station names.

## Requirements

* Wox
* Python 3 (set “Python Directory” on wox “general” settings)

## How to install

search `wpm install Y! Transit Search` on Wox.

or visit http://www.wox.one/plugin/319
